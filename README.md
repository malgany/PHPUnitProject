# PHPUnitProject

Neste projeto apliquei melhores práticas em testes unitários usadas no mercado e nas grandes startups.

## Installation

```bash
# clone o projeto:
git clone https://github.com/malgany/FlappyIA.git

#  rodar comando composer:
composer install

# executar:
phpunit src
# ou
./vendor/bin/phpunit src
```
